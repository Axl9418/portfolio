Weather API

Open Weather Map

This weather API project uses the Unity location service to request a location from the user. If it is not available, allow users to enter "latitude" and "longitude" coordinates.



I utilize the "OpenWeather API" response of the current weather description to change the skybox on the scene according to the information obtained.



In addition, you can click on the icon located in the right button corner to introduce new coordinates. Try North Pole coordinates to see snow!


You can test this project here: https://axlmonreal.tech/weather%20API/
