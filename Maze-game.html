Maze Game

Unity 3D game

The maze game is a first-person 3D game developed as a midterm project for the boot camp.

I implemented interfaces to allow the player to pick up and drop off items, such as a gun, and interact with other objects, like a button to activate the elevator platform.



I created a win and game-over scene to notify the player of the status of the game.

Finally, I made a simple UI to track the number of hidden items collected, see the remaining time on the clock, and health percentage on the screen.
